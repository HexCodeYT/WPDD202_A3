# Team Rules

1. Do not edit `css/style.css`, `css/responsive.css`, or `js/main.js` unless approved.
2. Only edit the page assigned to you.
3. Do not rename files.
4. Do not move folders.
5. Do not use templates.
6. Do not use AI-generated full pages.
7. Every image must have:
   - source URL
   - alt text
   - citation on references.html
8. Every page must keep the shared header and footer.
9. Make a branch before editing.
10. Open a pull request when finished.
